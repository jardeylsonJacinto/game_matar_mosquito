# Game Matar mosquito

A ideia do game é matar todos os mosquitos antes que o tempo acabe

## Sobre o projeto
Este projeto tem como proposta colocar em prática os conhecimentos adquiridos de css do curso de 
Desenvolvimento web da udemy.
